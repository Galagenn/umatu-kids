# UmaTu kids — статическая вёрстка

Обычная вёрстка без WordPress: HTML, CSS, JS.

## Как открыть

1. Открой папку `umatu-kids-verstka` в проводнике.
2. Дважды кликни по `index.html` — откроется главная в браузере.
3. Либо запусти локальный сервер из этой папки, например:
   - **VS Code:** расширение "Live Server", правый клик по `index.html` → Open with Live Server
   - **Node:** `npx serve .` и открой указанный адрес (например http://localhost:3000)

## Страницы

- **index.html** — главная (Hero, категории, о компании, форма «Запросить прайс-лист», портфолио)
- **catalog.html** — каталог «Интерактивные игровые терминалы» (сетка карточек, фильтр по типу)
- **portfolio.html** — портфолио с фильтром по отраслям
- **contacts.html** — контакты и форма «Заказать звонок», блок под карту
- **faq.html** — FAQ (аккордеон)

Стили в `css/style.css`, скрипты в `js/main.js`. Картинки в карточках — плейсхолдеры; при необходимости замени на тег `<img>` с нужным `src`.
